# 🚀 Node API Boilerplate

A simple and clean Node.js API boilerplate using Express.js.  
Perfect for quick projects, testing ideas, or building out scalable APIs.

---

## 📦 Features

- ✅ Express.js for building APIs
- 📁 Simple project structure
- 🧪 Easily extendable with routes & middleware
- 🧼 `.gitignore` and basic setup included
- 🧠 Designed for learning, interviews, and rapid prototyping

---

## 🛠 Tech Stack

- Node.js
- Express.js
